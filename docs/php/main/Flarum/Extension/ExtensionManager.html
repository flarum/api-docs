static array resolveExtensionOrder(Extension[] $extensionList)

<p>Sort a list of extensions so that they are properly resolved in respect to order.</p>

<p>Effectively just topological sorting.</p>

Parameters

Extension[] $extensionList

Return Value

array <p>with 2 keys: 'valid' points to an ordered array of \Flarum\Extension\Extension 'missingDependencies' points to an associative array of extensions that could not be resolved due to missing dependencies, in the format extension id =&gt; array of missing dependency IDs. 'circularDependencies' points to an array of extensions ids of extensions that cannot be processed due to circular dependencies</p>
