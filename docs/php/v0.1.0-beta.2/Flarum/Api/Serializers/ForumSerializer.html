protected callable getRelationship(string|Closure $serializer, string|Closure|null $relation = null, bool $many = false)

Get a closure that returns a Collection/Resource representing a relation.

Parameters

string|Closure $serializer The name of the serializer, or a Closure returning the name of the serializer, to use for the related items.
string|Closure|null $relation If a string is provided, it will be used to retrieve the relation data from the model: - If the relation is being included, the relation will be accessed as a property on the model. - If the relation is not being included and is a to-many relation, a list of IDs will be accessed as a property on the model with the suffix '_ids', otherwise by querying the relation method. - If the relation is not being included and is a to-one relation, the ID will be accessed as a property on the model with the suffix '_id'. If a closure is provided, it will be passed the model and whether or not the relation is being included. It is expected to return the relation data.
bool $many Whether or not this is a to-many relation.

Return Value
