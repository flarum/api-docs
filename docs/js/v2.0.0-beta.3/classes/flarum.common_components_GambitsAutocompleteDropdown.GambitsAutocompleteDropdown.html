    Class GambitsAutocompleteDropdown<CustomAttrs>

    This is an autocomplete component not related to the SearchModal forum components. It is a standalone component that can be reused for search inputs of any other types of resources. It will display a dropdown menu under the input with gambit suggestions similar to the SearchModal component.
