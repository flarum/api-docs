permissionGroups(callable $callable)

Dynamically process a user's list of groups when calculating permissions.

This can be used to give a user permissions for groups they aren't actually in, based on context. It will not change the group badges displayed for the user.

Parameters

callable $callable The callable can be a closure or invokable class, and should accept: - \Flarum\User\User $user: the user in question. - array $groupIds: an array of ids for the groups the user belongs to.</p> <p>The callable should return: - array $groupIds: an array of ids for the groups the user belongs to.
