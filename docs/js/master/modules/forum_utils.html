    • replyAction(goToLast: boolean, forceRefresh: boolean): Promise<void>
      • Defined in src/forum/utils/DiscussionControls.js:182

      Open the reply composer for the discussion. A promise will be returned, which resolves when the composer opens successfully. If the user is not logged in, they will be prompted. If they don't have permission to reply, the promise will be rejected.

      Parameters

      • goToLast: boolean

        Whether or not to scroll down to the last post if the discussion is being viewed.

      • forceRefresh: boolean

        Whether or not to force a reload of the composer component, even if it is already open for this discussion.

slidable

  • slidable(element: HTMLElement | Element | SVGElement): { reset: any }
    • Defined in src/forum/utils/slidable.js:15

    The slidable utility adds touch gestures to an element so that it can be slid away to reveal controls underneath, and then released to activate those controls.

    It relies on the element having children with particular CSS classes.

    The function returns a record with a reset proeprty. This is a function which reverts the slider to its original position. This should be called, for example, when a controls dropdown is closed.

    Parameters

    • element: HTMLElement | Element | SVGElement
