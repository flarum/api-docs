ClientView  

class ClientView implements Renderable

This class represents a view which boots up Flarum's client.

Properties

protected User $actor The user who is using the client.
protected null|string $title The title of the document, displayed in the <title> tag.
protected string $content The SEO content of the page, displayed in <noscript> tags.
protected string $layout The path to the client layout view to display.
protected null|array|object $document An API response that should be preloaded into the page.
protected array $variables Other variables to preload into the page.
protected array $bootstrappers An array of JS modules to import before booting the app.
protected array $headStrings An array of strings to append to the page's <head>.
protected array $footStrings An array of strings to prepend before the page's </body>.
